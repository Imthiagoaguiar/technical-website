# Technical Website

Website de exemplo para uma empresa de tecnologia, desenvolvido com React e Vite.

## Tecnologias

- [React 18](https://react.dev/)
- [Vite](https://vitejs.dev/)
- [Three.js](https://threejs.org/)
- HTML / JavaScript / JSON

## Como rodar localmente

```bash
# Instalar dependências
npm install

# Rodar em modo de desenvolvimento
npm run dev

# Gerar build de produção
npm run build

# Visualizar build de produção
npm run preview
```

## Observações

Este projeto é um exemplo/template de site para empresa tech. Não representa uma empresa real.
